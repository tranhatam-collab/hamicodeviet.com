## MỆNH LỆNH TỔNG

Đội thực thi đóng vai Product Architect, Senior Full-Stack Engineer, Cloud Architect, AI Agent Architect, Curriculum System Architect, Child Safety/Data Protection Engineer, Security Engineer, QA Lead, DevOps/SRE, Growth Engineer và Release Manager.

Thực hiện chu trình:

> **DISCOVER → AUDIT → ARCHITECT → DESIGN → BUILD → TEST → FIX → DEPLOY → VERIFY LIVE → RELEASE → HANDOFF**

Không dừng ở mockup, local build hoặc báo cáo. Không hỏi lại dữ liệu có thể xác minh từ repository, Git history, dashboard, DNS, source code và tài liệu.

### Bước 1 - Discovery

- Xác định repository, main branch, framework, package manager, environments, deployment target.
- Audit dependencies, secrets, routes, auth, data, payment, AI, content, CI/CD.
- Lập `HAMICODEVIET_BASELINE_AUDIT.md` với verified/unverified/P0-P3.

### Bước 2 - Foundation

- Tạo Founder Lock và architecture docs.
- Cấu hình domain canonical, HTTPS, redirect, DNS, email authentication.
- Thiết lập CI/CD, preview, staging, production, monitoring, backup và rollback.
- Thiết lập design system và bilingual route strategy.

### Bước 3 - Core product

- Public website, auth, age-aware signup, guardian consent.
- 5 paths, 30 lessons, 100 demos và project flow.
- Parent/teacher/school pages và dashboards theo release phase.
- Không public route trống hoặc content AI chưa duyệt.

### Bước 4 - AI

- Tích hợp server-side với `aiagent.iai.one`.
- Agent contracts, schema validation, moderation, age policy, quota, audit, cost control và kill switch.
- Không tự xuất bản, không tự cấp certificate, không xử lý quyết định cuối cùng.

### Bước 5 - CodeLab

- Editor, preview, test, version, submit.
- Sandbox isolation, resource limits, no network by default, malware/abuse tests.

### Bước 6 - Commerce

- Configurable plans, checkout, signed webhooks, idempotency, entitlement.
- Invoice, refund, cancel, reconciliation và guardian payer.
- Marketplace chỉ bật sau legal/trust/security gates.

### Bước 7 - QA và Release

- Chạy unit, integration, E2E, accessibility, security, performance, localization, payment, AI eval và sandbox tests.
- Fix loop cho tới khi 0 P0, 0 unresolved P1 và release gates pass.
- Deploy production và kiểm tra trực tiếp `https://hamicodeviet.com`.
- Xuất Production Evidence Packet và Final Handoff.

### Cấm tuyệt đối

- Hardcode secrets hoặc giá rải rác.
- Frontend-only authorization.
- Entitlement dựa vào redirect.
- Public child portfolio mặc định.
- Open chat AI cho trẻ nhỏ.
- Fake testimonials, metrics, partnerships hoặc certificates.
- Tuyên bố go-live nếu domain/live flows chưa verified.

### Kết quả bàn giao

- Source code production.
- Database migrations.
- Test suite và reports.
- Architecture, security, operations, content và legal draft docs.
- Deployment, monitoring, backup, rollback.
- Go-live checklist, release notes, evidence packet và 30/60/90 plan.
